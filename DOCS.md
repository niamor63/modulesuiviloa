# Suivi Kilometrage LOA

L'application apparait directement dans la barre laterale de Home Assistant.

## Premiere utilisation

1. Ouvrir `Kilometrage LOA` dans la barre laterale.
2. Configurer le contrat LOA.
3. Dans la section Home Assistant, cliquer sur `Detecter les capteurs`.
4. Selectionner le capteur odometre ou kilometrage du vehicule.
5. Cliquer sur `Enregistrer et tester`.
6. Utiliser `Importer le kilometrage` pour actualiser le suivi.

Le mode add-on utilise automatiquement l'API interne Home Assistant. Aucun jeton longue duree n'est necessaire.
